1. - Объект - Автомобиль.
   - Состояние :
        6-ти ступенчатая АККП.
   - Поведение :
        стоять на месте (при использовании этого поведения необходимо перевести рукоятку АККП в положение Р-парковка.)
        двигаться вперед (при использовании этого поведения необходимо перевести рукоятку АККП в положение D-движение, при нажатии педали газа объект начнет движение с автоматическим переключением передач.)
        двигаться назад (при использовании этого поведения необходимо перевести рукоятку АККП в положение R-передача заднего хода,при нажатии педали газа объект начнет движение назад.)

2. - Объект - Телефон.
   - Состояние :
        количество заряда батареи
        текущая яркость экрана
   - Поведение :
        увеличить яркость (при увеличении яркости уровень заряда батареи значительно быстрее начинает снижаться.)
        уменьшить яркость (при уменьшении яркости уровень заряда батареи сохраниться дольше.)

3. - Объект - Автомат.
   - Состояние :
        заряжен/не заряжен
        в боевом положении/на предохранителе
        холодный/горячий
   - Поведение :
        стреляет (при использовании этого поведения объект нагревается, уменьшается колличество патронов.)
        
4. - Объект - африканский ежик.
   - Состояние :
        голоден
        уставший/отдохнувший
   - Поведение :
        кушает корм (при этом поведении объект перестает быть голодным.)
        бегает в беговом колесе ( при этом поведении устает.)
        спит ( при этом поведении отдыхает, но снова становится голодным.)

5. - Объект - Столовая ложка.
   - Состояние :
        чистая/испачканая
        холодная/горячая
   - Поведение :
        кушать ложкой (при этом поведении объект становиться грязным)
        мыть в горячей воде ( при этом поведении объект становиться чистым и горячим.)

6. - Объект - Лед.
   - Состояние :
        масса
   - Поведение :
        находиться в тепле ( при этом поведении появляется вода, которая со временем испаряется, уменьшается масса объекта.) 

7. - Объект - Лампочка.
   - Состояние :
        вкл. выкл.
        температура
   - Поведение :
        включить (при этом поведении объект становиться ярким и нагревается.)
        выключить (при этом поведении - темнеет, остывает.)

8. - Объект - Тюбик клей момент.
   - Состояние :
        жидкий/твердый
        кол-во клея
   - Поведение :
        клеить (при этом поведении объект уменьшает количество клея, который становиться твердым.) 

9. - Объект - Носки.
   - Состояние :
        новые/использованные
        целые/рваные
   - Поведение :
        купить (при этом поведении объект новый.)
        надеть (при этом поведении объект используется и при длительной эксплуатации становиться рваным.)

10. - Объект - Компьютер.
    - Состояние :
         вкл. выкл.
         оперативная память
    - Поведение :
         включить (при этом поведении запускаются многие процессы оперативная память заполняется.)
